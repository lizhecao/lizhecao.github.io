iterm2 批量修改profiles

李哲操
2018-11-27
软件折腾
wiki, zsh

iterm2 支持批量修改profiles，只不过这个功能比较难找到，备个案

先修改其中一个profile, 然后如下操作:
preferences -> profiles -> 选中已经修改的profile -> (左下角) Other Actions -> Bulk copy from Selected Profile -> 勾选你要复制的配置(例如keys) -> 下方选中要批量修改的profiles -> 点击copy即可
